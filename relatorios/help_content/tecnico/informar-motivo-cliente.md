# Como informar motivo da viagem por cliente

## Quando usar
Use sempre que houver clientes vinculados ao relatório.

## Passo a passo
- Após selecionar os clientes, localize a área Motivo da viagem por cliente.
- Preencha um motivo claro para cada cliente.
- Revise antes de enviar para conferência.

## Observações importantes
O motivo por cliente é obrigatório no envio para conferência e substitui o motivo geral antigo.

## Erros comuns
- Informar motivo para apenas um cliente em relatório multi-cliente.
- Usar texto genérico demais para justificar a viagem.
