# Como selecionar clientes

## Quando usar
Use para informar quais clientes participaram da viagem ou atendimento.

## Passo a passo
- Na aba Dados gerais, clique em Selecionar clientes.
- Pesquise pelo nome do cliente.
- Marque todos os clientes envolvidos.
- Confirme a seleção e revise os badges exibidos na tela.

## Observações importantes
Todo cliente selecionado deve participar de pelo menos uma despesa ou trecho KM.

## Erros comuns
- Selecionar cliente que não terá movimento financeiro.
- Vincular uma despesa a cliente que não está no relatório.
