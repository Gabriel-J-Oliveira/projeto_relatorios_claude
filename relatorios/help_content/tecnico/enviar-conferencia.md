# Como enviar para conferência

## Quando usar
Use quando o relatório estiver completo e pronto para análise financeira.

## Passo a passo
- Revise dados gerais, clientes, motivos, despesas e KM.
- Corrija alertas exibidos no resumo.
- Clique em Enviar para conferência.
- Aguarde a análise do financeiro.

## Observações importantes
O backend valida itens válidos, clientes com movimento, motivos por cliente e integridade financeira.

## Erros comuns
- Relatório vazio.
- Cliente sem participação em despesa ou KM.
- Item sem cliente vinculado.
