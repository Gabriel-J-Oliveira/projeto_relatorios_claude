# Como informar KM excedente interno

## Quando usar
Use para deslocamentos internos, como hotel para cliente, cliente para restaurante ou evento para hotel.

## Passo a passo
- Na aba KM, localize KM excedente / deslocamento interno.
- Informe a quilometragem total.
- Preencha a observação explicando o deslocamento.
- Revise o resumo financeiro.

## Observações importantes
Em relatório multi-cliente, o KM excedente é dividido igualmente entre os clientes do relatório.

## Erros comuns
- Usar KM excedente para substituir trechos principais.
- Informar valor negativo.
