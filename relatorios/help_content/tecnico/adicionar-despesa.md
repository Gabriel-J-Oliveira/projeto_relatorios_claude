# Como adicionar despesa

## Quando usar
Use para registrar alimentação, hospedagem, pedágio, estacionamento ou outras despesas da viagem.

## Passo a passo
- Acesse a aba Despesas.
- Clique em Adicionar despesa.
- Informe data, tipo, descrição e valor.
- Selecione os clientes envolvidos naquela despesa.
- Anexe o comprovante quando aplicável.

## Observações importantes
Em despesas multi-cliente, o sistema gera rateio financeiro para conferência.

## Erros comuns
- Informar valor negativo.
- Não selecionar cliente da despesa.
- Esquecer comprovante obrigatório para análise.
