# Como gerar PDFs

## Quando usar
Use em relatórios finalizados, conforme disponibilidade dos botões.

## Passo a passo
- Abra a consulta final do relatório.
- Gere o PDF interno quando necessário.
- Gere PDFs por cliente ou o pacote ZIP.
- Confira os arquivos antes de encaminhamentos externos.

## Observações importantes
PDFs de cliente devem mostrar somente os dados do cliente correspondente.

## Erros comuns
- Tentar gerar PDF de cliente para relatório rejeitado.
