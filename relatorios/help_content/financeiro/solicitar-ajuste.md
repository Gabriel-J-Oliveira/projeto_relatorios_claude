# Como solicitar ajuste

## Quando usar
Use quando o relatório pode ser corrigido pelo técnico.

## Passo a passo
- Clique em Solicitar ajuste.
- Informe uma justificativa objetiva.
- Confirme a ação.
- O relatório retorna para o técnico.

## Observações importantes
Prefira solicitar ajuste quando faltarem dados, comprovantes ou explicações.

## Erros comuns
- Rejeitar definitivamente quando uma correção simples resolveria.
