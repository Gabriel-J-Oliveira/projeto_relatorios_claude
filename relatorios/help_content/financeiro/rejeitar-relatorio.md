# Como rejeitar relatório definitivamente

## Quando usar
Use quando o relatório não deve ser aprovado nem devolvido para correção.

## Passo a passo
- Clique em Rejeitar relatório.
- Informe a justificativa obrigatória.
- Confirme a rejeição.

## Observações importantes
A rejeição definitiva encerra o relatório e impede edição.

## Erros comuns
- Usar rejeição definitiva para problemas corrigíveis.
