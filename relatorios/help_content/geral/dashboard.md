# Como usar o dashboard

## Quando usar
Use para acompanhar indicadores e movimentações recentes.

## Passo a passo
- Ajuste filtros de período, cliente, técnico ou status.
- Leia os KPIs do topo.
- Analise gráficos e relatórios recentes.
- Use Ver guia para refazer o tour da tela.

## Observações importantes
Técnicos veem apenas seus próprios dados. Financeiro e administradores veem visão global.

## Erros comuns
- Comparar dados com filtros diferentes.
- Esperar ver dados globais usando perfil técnico.
