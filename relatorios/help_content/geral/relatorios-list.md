# Como consultar relatórios

## Quando usar
Use para localizar relatórios, acompanhar status e acessar ações disponíveis.

## Passo a passo
- Acesse Relatórios.
- Use filtros e busca rápida.
- Clique no número para ordenar.
- Use os botões de ação conforme status e permissão.

## Observações importantes
Relatórios aprovados ou rejeitados abrem na tela de consulta final.

## Erros comuns
- Procurar por relatório fora do período filtrado.
- Tentar editar relatório já finalizado.
